# Intelligent & Communicating Systems Labs

Short collection of ICS lab reports covering microcontrollers, Arduino GPIO, sensors, Raspberry Pi communication, IoT platforms, MQTT/Node-RED, and a final ESP32 Tuya/Home Assistant project.

## Contents

- **Lab 01 - Microcontroller Basics:** Von Neumann vs. Harvard architectures, LEDs, pushbuttons, and Arduino MKR1010 hardware.
- **Lab 02 - Arduino GPIO, Sensors & Actuators:** Digital/analog GPIO, debounce circuits, and pull-up/pull-down resistors.
- **Lab 03 - Force Sensor & Interrupts:** FSR voltage-divider readings and Arduino interrupt service routines.
- **Lab 04 - PWM & Raspberry Pi Intro:** PWM output plus Arduino vs. Raspberry Pi GPIO comparison.
- **Lab 05 - Wired Communications:** UART and I2C communication between Arduino and Raspberry Pi, including ADS1115 ADC usage.
- **Lab 06 - IoT Platform Basics:** Cloud vs. on-premise IoT platforms, comparing Blynk, Arduino Cloud, Home Assistant, and SmartThings.
- **Lab 08 - MQTT & Node-RED:** Node-RED visual flows, MQTT publish/subscribe, and comparison with OpenHAB.
- **Project Report:** ESP32 IoT device control with Tuya, Home Assistant, local MQTT, and a water level monitoring system.
- **Supplementary ESP32 Tuya Guide:** Step-by-step Tuya Developer Console setup for ESP32 custom firmware flashing.
